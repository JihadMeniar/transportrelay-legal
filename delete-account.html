Supprimer mon compte

Vous pouvez demander la suppression définitive de votre compte et de toutes vos données personnelles associées.

⚠ Attention : Cette action est irréversible. Une fois votre compte supprimé, vous ne pourrez pas récupérer vos données, votre historique de courses ou votre abonnement.

Données supprimées définitivement :

  • Profil (nom, email, téléphone)
  • Historique des courses publiées et acceptées
  • Messages et pièces jointes du chat
  • Tokens de notification push
  • Code de parrainage et filleuls associés

Données conservées :

  • Données de facturation Stripe (obligation légale 10 ans)
